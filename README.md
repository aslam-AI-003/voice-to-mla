## How to View

Open in browser:
- **Hub:** `voice-to-mla/index.html`
- **Portfolio:** `voice-to-mla/portfolio/index.html`
- **Demo:** `voice-to-mla/demo/index.html`

Or run a local server:
```bash
cd voice-to-mla
python3 -m http.server 8080
```
Then open: http://localhost:8080
